# DevOps 实践

总结 DevOps 学习及实践的经验及问题的解决。

## 一、变革清单

### （1）组织变革

* 从职能制组织，变革为弱矩阵式组织

### （2）流程变革

* 从瀑布式研发流程，变革为持续集成流程

### （3）基础设施变革

* 从 ClearCase 到 Git
* 从 Jenkins 到 Gitlab

### （4）产品规划变革

* 从多版本演进，变革统一版本、多分特性分支

## 二、待解决的挑战

* 异构平台的支持
* 持续集成的由浅入深
* 信息的高效共享与流转
* 业务范围的持续拓展
